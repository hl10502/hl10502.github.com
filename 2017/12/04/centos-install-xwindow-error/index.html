发表于 2017-12-04 | 分类于 CentOS | 阅读次数

解决方法

其实解决方法比较显然，根据提示一步步操作即可：

输入1，回车显示License信息

根据提示信息，输入2，接受授权

键入q，离开设置;

输入yes，保存信息即可进入CentOS7.2的登录界面。
