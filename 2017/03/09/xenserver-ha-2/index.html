XenServer HA Internal error: Xapi_ha.Xha_error(15)

发表于 2017-03-09 | 分类于 XenServer ， HA | | 阅读次数

XenServer资源池启用HA出现内部错误 Internal error: Xapi_ha.Xha_error(15)

这是因为xha进程已经启动了，启用失败。

日志分析

查看日志出错信息

启用HA过程

启用HA时，创建心跳盘，会初始化pool的状态，检测是否启动ha守护进程，如果已经存在则会报错：

Could not set the pool state to INIT because the HA daemon may be present

不存在，就会启动xha进程，监控池内主机是否发生心跳信号（存储检测、网络检测）

解决方法

查找xhad进程，进程存在会生成一个/etc/xensource/xhad.conf配置文件，记录HA的心跳检测、主机等配置信息

kill进程

重新启用HA，同时指定作为信号检测的存储池

# XenServer # HA # 资源池高可用
XenServer HA
