虚拟机VDI扩容

发表于 2017-03-16 | 分类于 XenServer ， VDI | 阅读次数

虚拟机的磁盘容量不够时，可以进行扩容操作。
两种方式：

  • 增加新的虚拟磁盘,比较简单，直接添加数据盘通过vdi-create、vbd-plug就可以。
  • 扩展磁盘的大小，增大磁盘的容量。

下面以Centos7为例，扩展系统盘的大小从10G扩展到50G。
