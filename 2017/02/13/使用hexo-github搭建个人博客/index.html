使用hexo+github搭建个人博客

发表于 2017-02-13

使用hexo+github在windows7环境下搭建个人技术博客,在git bash下执行以下bash操作命令。

环境准备

  • windows7 x64
  • nodejs 6.9.5
  • git 2.11.0
  • hexo 3.2.2

配置github

创建仓库

（1）打开github主页，新建github仓库，仓库名称为”hl10502.github.com”

配置SSH key

（1）生成key

（2）打开github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：

将(1)中生成的key文件（用户目录下的.ssh\id_rsa.pub）内容copy到新建的key中

（3）测试SSH配置

（4）配置全局的用户名、邮箱

配置hexo

（1）安装hexo（注意：之前安装的nodejs在安装时需要添加环境变量）

（2）在F盘下创建 F:\blog\hexo文件夹，作为blog代码目录，初始化hexo

（3）下载next主题

（4）修改站点的_config.yml文件的主题为
theme: next

（5）生成静态文件

（4）启动hexo服务，可以通过 http://localhost:4000/访问

配置hexo-hey插件

使用git bash来执行以下命令

（1）安装hexo-hey插件

（2）配置站点的_config.yml文件，添加admin部分

#hexo-hey插件
admin:
    name: hexo
    password: hey
    secret: hey hexo
    expire: 60*1
    # cors: http://localhost:4000

（3）登录 http://localhost:4000/admin，可以新建文章，发布到本地生成.md文件

用户名：hexo
密码：hey

发布hexo到github

（1）安装hexo-deployer-git插件

（2）配置站点的_config.yml文件，修改deploy部分，使用SSH上传文件

deploy:
  type: git
  repository: git@github.com:hl10502/hl10502.github.com.git
  branch: master

hl10502为github的用户名，hl10502.github.com为仓库名称

（3）部署hexo，上传到github

（4）访问个人博客： https://hl10502.github.io
